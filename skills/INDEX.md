## 📝 Skill Structure

Each skill has:
```
skill-name/
├── skill_name_v1/
│   └── SKILL.md       # Full skill documentation
└── VERSIONS.md        # Version history
```

### SKILL.md Template:
```markdown
# Skill Name

## Overview
## When to Use
## When NOT to Use
## Core Concepts
## Architecture Patterns
## Best Practices
## Anti-Patterns
## Examples
## Tools & Libraries
## Metrics to Track
## Checklist
```
